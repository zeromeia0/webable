Local-first

Webable

Personal finance app that runs on your computer. This GitHub Pages site is only a quick install guide — not the application itself.

What runs where

GitHub Pages hosts this documentation page only.

The real app is a Python (FastAPI) server you run with Docker on your machine.

Your data (SQLite databases, uploaded bank PDFs, caches) stays in a data/ folder next to the project on your disk — not in the cloud.

What you need

  • Git
  • Docker
  • Docker Compose (often included with Docker Desktop)

Install and start

In a terminal:

git clone https://github.com/zeromeia0/webable.git
cd webable
docker compose up -d --build

Then open the app in your browser:

http://localhost:8080

Stop

docker compose down

Restart

docker compose restart

Update

git pull
docker compose up -d --build

View logs

docker compose logs -f

Backup

Copy the entire data/ folder somewhere safe (external drive, encrypted archive, etc.). That folder holds your app database and uploads.

Warning: Do not delete data/ unless you intend to remove your local Webable data.

Privacy reminder

Webable is intended for local use. Do not expose it to the public internet without authentication, HTTPS, a reverse proxy, and a proper security review.
